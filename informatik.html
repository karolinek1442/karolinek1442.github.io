Datatyper:
Number -> nummer Datatype

Objekt -> er en speciel form for variabel, hvor der er flere værdier adskilt af kommaer.
Boolean -> kan enten have værdien true eller false

Comparison operators -> viser om en værdi er mindre eller større end en anden værdi. Med "=="

Assignment operators -> tildeler og plusser med værdier.

Logical operators -> sætter udtryk sammen, og afgør om det er sandt. (hvis en af udtrykkene er sande med and, or eller not.) man kan også bruge ! som betyder "not".

Arithmetic operator -> løser regnestykker.

If statement -> tjekker om betingelser er opfyldt.

Funktioner -> bruger man til at gennem stykker af logik.

Man bruger CSS til at difinere hvordan en hjemmeside skal se ud.

Man bruger HTML til at difinere indholdet på en hjemmeside

1

Contrast størelse -> fx overskrift er større end normal tekst.
Contrast bredde -> fx om teksten er fed tekst.
Contrast farve -> fx bagrund sort forgrund hvid.
Contrast lys -> det skal være tydligt at fx læse tekst på hjemmeside.
Juxtaposition -> fx skabe opmærksomhed på siden.

2
Repitation -> fx forskellige billeder. De sammen ting må ikke gå igen for mange gang.

3
Alignment -> positionere ting ordentligt. Fx ting skal være på sammen linje.

4
Proximity -> tæthed. Ting der står tæt på hindanden hører sammen. (Lave afsnit)

5
White space -> Tomt rum. Fx google. Drage opmærksomhed mod objekt. Man er ikke i tvivl om hvad man skal på hjemmesiden.
